# Task Flow CLI

Um gerenciador de tarefas simples e eficiente rodando diretamente no terminal, construído em Python.

## Funcionalidades
- Criar tarefas com níveis de prioridade (Alta, Média, Baixa).
- Listagem organizada com ícones de status.
- Marcar tarefas como concluídas.
- Excluir tarefas com validação de índice.
- Interface amigável com pausas dinâmicas.

## Estrutura do Projeto
- `main.py`: Ponto de entrada do sistema.
- `tarefas.py`: Lógica de manipulação dos dados.
- `menu.py`: Interface visual do terminal.

## Como rodar
1. Certifique-se de ter o Python instalado.
2. Clone o repositório.
3. Execute: `python main.py`
